# Fabric Mod — Phase 3

This folder will contain the Fabric mod for advanced villager AI.

**Do not start this until Phase 1 is complete and fun.**

## Planned Features
- Custom villager Goal classes (MoveToWorksite, Patrol, Build, Gather)
- Villager memory system (NBT-based)
- Blueprint building (reads .nbt structure files)
- Advanced pathfinding
- Village resource economy

## Setup (when ready)
1. Install Fabric Mod development template
2. Target Minecraft 1.21.x + Fabric Loader
3. Use `fabric-loom` Gradle plugin
4. Namespace: `com.aelori.mod`
